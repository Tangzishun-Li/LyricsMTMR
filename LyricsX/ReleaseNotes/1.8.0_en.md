# LyricsX 1.8.0 What's New

## New Features

- **Musixmatch lyrics source** — Fetch lyrics from Musixmatch by entering your personal token in Preferences > Lab.
- **Lyrics source priority** — A new "Source" preferences tab lets you rank lyrics providers by priority.
- **Write plain LRC to iTunes/Music** — New option to save lyrics in plain LRC format (without karaoke time tags) to your Apple Music library.
- **macOS 15.4 MediaRemote compatibility** — Starting with macOS 15.4, the system enforces entitlement checks on MediaRemote calls. LyricsX now bypasses this restriction to keep now-playing detection working on newer systems.

## Improvements

- **More reliable lyrics search** — Fixed a race condition where stale search results could overwrite the current track's lyrics. Searches are now properly cancelled on track change.
- **Better lyrics ranking** — Fixed sorting that incorrectly favored whichever result arrived first. Results are now ranked by quality and source priority.
- **Local lyrics loading** — Embedded lyrics in music files now take priority over external lyrics files.
- **Kugou & NetEase improvements** — Better search results from Kugou and NetEase lyrics providers.

## System Requirements

- Minimum requirement: **macOS 11.0 (Big Sur)**
