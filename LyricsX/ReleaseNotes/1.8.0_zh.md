# LyricsX 1.8.0 更新内容

## 新功能

- **新增 Musixmatch 歌词源** — 支持通过 Musixmatch 获取歌词，可在偏好设置的 Lab 页签中填入个人 Token 启用
- **歌词来源优先级设置** — 新增「歌词来源」设置页签，可自定义各歌词源的优先级排序，让偏好的来源优先匹配
- **写入传统 LRC 到 iTunes/Music** — 新增选项，可将歌词以纯 LRC 格式（不含逐字卡拉 OK 时间标签）写入 Apple Music 曲库
- **适配 macOS 15.4 MediaRemote 限制** — 从 macOS 15.4 起，系统对 MediaRemote 调用新增了 entitlement 验证，LyricsX 已绕过此限制，确保在新系统上仍能正常检测正在播放的音乐

## 改进

- **歌词搜索更稳定** — 修复了切换歌曲时旧搜索结果可能覆盖新结果的问题，现在切歌后会正确取消上一次搜索
- **歌词排序更准确** — 修复了歌词排序始终优先显示最先收到结果的问题，现在会按照质量和来源优先级正确排序
- **本地歌词加载优化** — 嵌入在音乐文件中的歌词现在优先于外部歌词文件加载
- **酷狗、网易云歌词源增强** — 改善了酷狗和网易云音乐歌词源的搜索效果

## 系统要求

- 最低系统要求：**macOS 11.0 (Big Sur)**
