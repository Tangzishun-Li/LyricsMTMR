# v1.8.5

1.8.4 的热修复重发版本。1.8.4 打 tag 时依赖库尚未打上对应的 tag，导致发布的二进制并未真正带上修复。1.8.5 在已正确打 tag 的依赖之上重新构建；损坏的 1.8.4 release 已从 GitHub 删除。

## 修复

- **修复 macOS 10.15 (Catalina) 上启动崩溃** — 修正了应用在 macOS 10.15 启动时崩溃的问题。修复在 MusicPlayer 1.8.4 中，1.8.5 通过刷新后的 SPM 锁文件引入该版本

## 系统要求

- 最低系统要求：**macOS 10.15 (Catalina)**
