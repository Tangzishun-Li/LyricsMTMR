# v1.8.5

A hotfix re-release of 1.8.4. The 1.8.4 build was tagged before its
dependency libraries had matching tags, so the published binary did
not actually carry the fix. 1.8.5 rebuilds on top of properly tagged
dependencies; the broken 1.8.4 release has been removed from GitHub.

## Fixes

- **Fixed startup crash on macOS 10.15 (Catalina)** — Resolved an issue
  that caused the app to crash on launch on macOS 10.15. The fix ships
  in MusicPlayer 1.8.4, which 1.8.5 now picks up via the refreshed SPM
  lockfile.

## System Requirements

- Minimum requirement: **macOS 10.15 (Catalina)**
