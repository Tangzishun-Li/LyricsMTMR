# LyricsX 1.8.3 What's New

## New Features

- **macOS 10.15 (Catalina) support restored** — LyricsX 1.8.3 once again runs on macOS 10.15. **Note: 1.8.3 is expected to be the final release that supports 10.15; future versions may drop it.**
- **NowPlaying support for iOS apps on Mac** — Now-playing detection now correctly identifies iOS apps running on Apple Silicon Macs as a NowPlaying source.

## Fixes

- **Fixed "Launch with Player" no longer triggering** — After upgrading, please toggle the "Launch with Player" preference off and on again to re-arm the helper.

## Other Changes

- **Main app no longer sandboxed** — Upcoming features cannot run inside the App Sandbox, so the main app has dropped the sandbox entitlement. LyricsX is fully open source — you can audit the entire codebase at https://github.com/MxIris-LyricsX-Project/LyricsX.

## System Requirements

- Minimum requirement: **macOS 10.15 (Catalina)**
